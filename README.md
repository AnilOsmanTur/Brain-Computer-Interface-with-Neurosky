# Brain-Computer-Interface-with-Neurosky

Introduction
------------
Brain Computer Interface (BCI) with Neurosky Mindwave Mobile 2 that enables anyone to use a computer, mobilephone etc.
 

Requirements
~~~~~~~~~~~~
 - NeuroPy [link](https://github.com/lihas/NeuroPy)
 - numpy
 - Kivy
